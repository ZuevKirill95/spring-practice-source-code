# React-demo

## Запуск проекта

1. Установите [node.js](https://nodejs.org/en).
2. Зайдите в папку `react-demo` через терминал.
3. Выполните команду `npm i`, чтобы скачать библиотеки.
4. Выполните команду `npm start`, чтобы запустить проект.
